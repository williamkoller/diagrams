# Diagrams 

### Required `python3` and `pip`

- add in `.zshrc` this line `alias py="python3"`

- `py -m venv diagrams-env`

- `source diagrams-env/bin/activate`

- `py -m pip install diagrams`

- `sudo apt-get install graphviz`

- `py diagram.py`
